## 📝 使用建议

### 日常使用

```bash
# 快速诊断
node dist/index.js check --fast

# 诊断特定服务器
node dist/index.js server <name> --fast

# 性能分析
node dist/index-optimized.js profile

# 缓存预热
node dist/index-optimized.js warmup
```

### 搜索功能

```bash
# MCP 包搜索
node dist/index.js search "ai cli" --limit 20

# Web 搜索
node dist/index.js web-search "query" --engine google
```

### 爬取功能

```typescript
import { EnhancedWebCrawler } from './dist/tools/enhanced-crawler.js';

const crawler = new EnhancedWebCrawler();
const result = await crawler.crawl('https://example.com', {
  maxPages: 10,
  concurrency: 3,
  extractMetadata: true,
});
```
