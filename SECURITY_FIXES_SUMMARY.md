## 🔴 发现并修复的问题

### 问题 1: @modelcontextprotocol/sdk 高危漏洞

**CVE 编号**: 
- GHSA-8r9q-7v3j-jr4g
- GHSA-w48q-cv73-mx4w

**严重程度**: 🔴 高危

**漏洞描述**:
1. **ReDoS (正则表达式拒绝服务攻击)**
   - 攻击者可通过恶意正则表达式导致服务拒绝
   
2. **DNS Rebinding 攻击**
   - 默认未启用 DNS rebinding 保护
   - 可能导致未授权访问

**影响版本**: <=1.25.1  
**修复版本**: 1.27.1

**修复命令**:
```bash
cd server
npm audit fix --force
```

**修复结果**:
```
added 71 packages, changed 1 package
found 0 vulnerabilities ✅
```

**状态**: ✅ 已修复
